AgentForge – Production-Ready Multi-Agent AI Platform
🚀 Overview

AgentForge is a production-ready Multi-Agent AI Platform built using the MERN Stack and modern AI technologies. It supports AI agents, Retrieval-Augmented Generation (RAG), vector search, document processing, streaming AI responses, authentication, and scalable microservices.
